# Celsius to Fahrenheit Converter App

A simple beginner Android app built using XML and Java.  
This app converts temperature from Celsius to Fahrenheit.

---

## Features

- Enter temperature in Celsius
- Click Convert button
- Displays temperature in Fahrenheit
- Clean and simple UI
- Built using Android Studio

---

## Requirements

Make sure you have:

- Android Studio installed
- Java JDK installed
- Android SDK installed (comes with Android Studio)

---

## How to Clone the Project (Using Bash)

Open Git Bash or Terminal and run:

git clone https://github.com/Lakshmisai3137/Temperature-Converter.git

Then go inside the project folder:

cd Temperature-Converter

---

## How to Open in Android Studio

1. Open Android Studio  
2. Click **Open**  
3. Select the cloned project folder  
4. Click **OK**  
5. Wait for Gradle Sync to finish  

---

## How to Run the App

1. Connect your Android device  
   OR  
   Open an Android Emulator  

2. Click the green **Run ▶** button  

3. Select your device  

4. App will install and start  

---

## How to Use the App

1. Enter temperature in Celsius  
2. Click the Convert button  
3. Fahrenheit value will be displayed  

Example:

Input: 25  
Output: 77  

---

## Project Structure

- activity_main.xml → UI Design (XML)
- MainActivity.java → App Logic (Java)
